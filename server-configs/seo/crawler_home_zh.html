    API Example (Python)

    from openai import OpenAI

client = OpenAI(
    api_key="your-enlyai-api-key",
    base_url="https://enlyai.com/v1"
)

response = client.chat.completions.create(
    model="enlyai-chat",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)

    API Example (cURL)

    curl https://api.enlyai.com/v1/chat/completions \
  -H "Authorization: Bearer your-enlyai-api-key" \
  -H "Content-Type: application/json" \
  -d '{"model":"enlyai-chat","messages":[{"role":"user","content":"Hello!"}]}'
